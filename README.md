# Banking Data Analytics Dashboard

This project focuses on analyzing a **Banking Dataset** and building an **interactive analytics dashboard** that provides insights into customers, transactions, channels, and currencies. The key objective is to transform raw banking data into meaningful visuals for decision-making.  

The dashboard enables users to quickly understand:  
- Total number of customers  
- Total transaction amounts (USD & EUR)

- Comparative insights across branches, cities, and currencies

## ğŸ› ï¸ Data Analysis Process  

The project was developed following the standard **Data Analysis Lifecycle**:  

1. **Data Collection**  
   - Sourced a transactional banking dataset containing customer details, transaction records, currencies, channels, and geographic information.  

2. **Data Cleaning & Preparation**  
   - Handled missing and inconsistent values  
   - Standardized currency values (USD, EUR)  
   - Verified branch-city consistency  
   - Ensured proper formatting of dates for time-series visuals  

3. **Data Transformation**  
   - Created derived metrics such as total customers, monthly totals, and transaction types  
   - Aggregated data by branches, channels, and cities  
   - Converted transaction columns into analysis-ready measures  

4. **Data Visualization & Dashboard Development**  
   - Designed intuitive card visuals to highlight **KPIs**:
     - Total Customers  
     - Total Transactions  
     - Total Transaction Amount  
     - Transaction by Mode  
   - Implemented advanced interactive features:
     - **Slicers** for Branch, City, and Currency (USD/EUR)  
     - **Donut Chart**: Total transaction amount segmented by year  
     - **Pie Chart**: Transaction amount distribution by channel  
     - **Stacked Column Chart**: Transaction amount by **Month Name**  
     - **Area Chart**: Total transaction amount by transaction type  

5. **Insights & Storytelling**  
   - Identified top-performing branches and cities  
   - Compared currency usage (USD vs EUR)  
   - Uncovered transaction trends across months and channels  
   - Highlighted customer behavior in transaction types over time
